# Rosas2015 - Caffeine-induced luminal SR calcium changes

This SBML model reproduced the calcium release from SR by application of 20 mM or 2mM caffeine, described in the paper. * Ca_i_Total and Ca_SR_Total respectively represent the total calcium concentrat.
