# Mitrophanov2013 - Simulation of Hockin Blood Coagulation Model under reduced temperature

Blood coagulation model using an updated Hockin2002 model. New reactions for factor X and V activation by IXa and mIIa respectively (reactions R28 and R29, parameters k43 and k44).
