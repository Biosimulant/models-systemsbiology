# Eungdamrong2007_Ras_Activation

The model reproduces the time profiles of Golgi Ras-GTP and plasma membrane Ras-GTP, subjected to a palmitoylation rate of 0.00015849 second inverse. This is depicted in Fig 5a and 5b for various palm.
