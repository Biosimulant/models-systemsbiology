# Zhou2015 - Blood coagulation model predicting effects of factor Xa inhibitors

Blood coagulation model investigating effects of Xa-inhibitors (Rivaroxaban and Apixaban). Model is an extension of Pohl1994 and reduced from Wajima2009.Encoding the model from the supplementary files.
