# Zeilinger2006_PRR7-PRR9light-Y

The model reproduces the time profile of cYm and cTm under light-dark cycles as depicted in Fig 4 and Fig 5 respectively. 12 hour light-dark cycles are accomplished using a simple algorithm in the eve.
