# Burghaus2014 - Simulating blood coagulation factor Xa inhibitors

Mathematical model of blood coagulation factors II, VII, IX and X as well as protein C and protein S and the effect of warfarin on kinetic rates. It can be used to explore systemsbiology burghaus2014 simulating blood coagulation factor dynamics and compare simulation behavior across conditions.
